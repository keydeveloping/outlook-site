📚 API Documentation

Authentication

Include your API key in the request header:

X-API-Key: your_api_key_here

Available Endpoints

List Accounts

GET /api/v1/agent/accounts

curl -H "X-API-Key: YOUR_KEY" http://localhost:5000/api/v1/agent/accounts

Get Inbox

GET /api/v1/agent/inbox/{email}

curl -H "X-API-Key: YOUR_KEY" http://localhost:5000/api/v1/agent/inbox/user@example.com

Get Message

GET /api/v1/agent/message/{email}/{message_id}

curl -H "X-API-Key: YOUR_KEY" http://localhost:5000/api/v1/agent/message/user@example.com/MSG123

Search Messages

GET /api/v1/agent/search/{email}?q={query}

curl -H "X-API-Key: YOUR_KEY" "http://localhost:5000/api/v1/agent/search/user@example.com?q=invoice"

Permission Levels

Read Only - Can list accounts and read messages

Read-Write - Can read and mark messages as read/unread

Admin - Full access including delete operations
